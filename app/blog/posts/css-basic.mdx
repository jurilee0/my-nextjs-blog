안녕하세요! 지난 포스팅에서는 HTML을 살펴보았습니다. 오늘은 구조에 예쁜 옷을 입히는 CSS에 대해 알아보겠습니다.

## 1️⃣ CSS란?

CSS는 'Cascading Style Sheets'의 약자로 웹 페이지의 스타일을 정의하는 언어입니다.
HTML 문서에 스타일을 적용하여 디자인을 추가하고 웹 페이지를 더욱 보기 좋게 만들어줍니다.

## 2️⃣ CSS의 기본 구조

CSS는 선택자(selector)와 선언(declaration)으로 구성됩니다.
선택자는 스타일을 적용할 HTML 요소를 선택하고, 선언은 속성(property)과 값(value)으로 구성됩니다.

```css
selector {
  property: value;
}
```

## 3️⃣ CSS 선택자

CSS 선택자는 HTML 요소에 스타일을 적용하기 위해 사용됩니다. 선택자를 통해 특정 요소를 선택하고, 그 요소에 스타일을 지정할 수 있습니다.

### 기본 선택자

- `*`: 모든 요소를 선택합니다. 주로 초기화 스타일을 적용할 때 사용됩니다.
- `tag`: 특정 태그를 선택합니다. 예를 들어, `div`, `p`, `span` 등 HTML 태그 이름을 사용합니다.
- `.classname`: 특정 클래스를 가진 요소를 선택합니다. 여러 요소에 동일한 스타일을 적용할 때 유용합니다.
- `#id`: 특정 ID를 가진 요소를 선택합니다. 페이지 내에서 유일한 요소에 스타일을 적용할 때 사용합니다.
- `[attribute]`: 특정 속성을 가진 요소를 선택합니다. 예를 들어, `input[type="text"]`는 텍스트 입력 요소를 선택합니다.

### 결합자

- `div p`: `div` 요소의 모든 자손 `p` 요소를 선택합니다.
- `div > p`: `div` 요소의 직계 자식 `p` 요소를 선택합니다.
- `h1 + p`: `h1` 요소 바로 다음에 오는 `p` 요소를 선택합니다.
- `h1 ~ p`: `h1` 요소 이후에 나오는 모든 `p` 요소를 선택합니다.

### 가상 클래스와 가상 요소

- `:hover`: 요소에 마우스를 올렸을 때 스타일을 적용합니다.
- `:focus`: 요소가 포커스를 받을 때 스타일을 적용합니다.
- `:nth-child(n)`: 부모 요소의 n번째 자식 요소를 선택합니다.
- `::before` 및 `::after`: 요소의 앞이나 뒤에 가상 요소를 삽입합니다. 주로 `content` 속성과 함께 사용됩니다.

### 선택자 우선순위

CSS 선택자 우선순위는 스타일 충돌 시 어떤 스타일이 적용될지를 결정합니다. 우선순위는 다음과 같습니다:

1. 인라인 스타일 (style 속성)
2. ID 선택자 (#id)
3. 클래스 선택자 (.class), 속성 선택자, 가상 클래스
4. 태그 선택자
5. 전체 선택자 (\*), 결합자 (>, +, ~)

## 4️⃣ CSS 속성

### 텍스트 관련

텍스트 스타일링은 웹 페이지의 가독성을 높이고, 디자인을 개선하는 데 중요한 역할을 합니다.

- `color`: 텍스트의 색상을 지정합니다. HEX, RGB, RGBA 등의 형식을 사용할 수 있습니다.
- `font-size`: 텍스트의 크기를 지정합니다. px, em, rem, %, vw 등의 단위를 사용할 수 있습니다.
- `font-family`: 텍스트의 글꼴을 지정합니다. 여러 글꼴을 쉼표로 구분하여 지정할 수 있습니다.
- `font-weight`: 텍스트의 굵기를 지정합니다. normal, bold, 100~900 등의 값을 사용할 수 있습니다.

### 박스 모델

박스 모델은 요소의 크기와 여백을 관리하는 데 사용됩니다.

- `width` 및 `height`: 요소의 너비와 높이를 지정합니다.
- `margin`: 요소의 외부 여백을 지정합니다. 네 방향을 개별적으로 지정할 수도 있습니다.
- `padding`: 요소의 내부 여백을 지정합니다. 네 방향을 개별적으로 지정할 수도 있습니다.
- `border`: 요소의 테두리를 지정합니다. 두께, 스타일, 색상을 설정할 수 있습니다.
- `box-sizing`: 요소의 크기 계산 방식을 지정합니다. `content-box`와 `border-box`가 있습니다.

### 레이아웃

레이아웃 속성은 요소의 위치와 배치를 제어합니다.

- `position`: 요소의 위치를 지정합니다. `static`, `relative`, `absolute`, `fixed`, `sticky` 등이 있습니다.
- `z-index`: 요소의 쌓임 순서를 지정합니다. 값이 클수록 위에 표시됩니다.
- `float`: 요소를 왼쪽 또는 오른쪽으로 띄워 배치합니다.
- `clear`: 떠 있는 요소의 영향을 받지 않도록 합니다.

### 배경

배경 속성은 요소의 배경을 설정하는 데 사용됩니다.

- `background`: 배경 색상, 이미지, 반복, 위치, 크기를 축약형으로 설정합니다.
- `background-color`: 배경 색상을 지정합니다.
- `background-image`: 배경 이미지를 지정합니다.
- `background-position`: 배경 이미지의 위치를 지정합니다.
- `background-size`: 배경 이미지의 크기를 지정합니다.

## 5️⃣ 반응형 웹 디자인

반응형 웹 디자인은 다양한 화면 크기와 장치에 적응하는 웹 페이지를 만드는 데 사용됩니다.

### 미디어 쿼리

미디어 쿼리는 특정 조건에 따라 CSS 스타일을 적용할 수 있게 합니다.

```css
@media screen and (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}
```

### CSS 변수 활용

CSS 변수는 재사용 가능한 값을 정의하고 사용할 수 있게 합니다.

```css
:root {
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
}

.button {
  background-color: var(--primary-color);
}
```

## 6️⃣ CSS 방법론

CSS 방법론은 CSS를 효율적으로 관리하고 유지보수하기 위한 규칙과 가이드라인입니다.

### BEM (Block Element Modifier)

BEM은 블록, 요소, 수정자로 구성된 명명 규칙을 사용하여 CSS를 구조화합니다.

```css
/* Block */
.card {
}

/* Element */
.card__title {
}
.card__image {
}

/* Modifier */
.card--featured {
}
.card--dark {
}
```

### OOCSS (Object-Oriented CSS)

OOCSS는 구조와 스킨을 분리하여 재사용성을 높이는 방식입니다.

```css
/* 구조 */
.button {
  padding: 10px 20px;
  border-radius: 4px;
}

/* 유형별 스타일링 */
.button-primary {
  background: blue;
  color: white;
}
```

### SMACSS (Scalable and Modular Architecture for CSS)

SMACSS는 CSS를 모듈화하여 확장성과 유지보수성을 높이는 방법론입니다.

```css
/* Base */
body {
}

/* Layout */
.l-header {
}

/* Module */
.nav {
}

/* State */
.is-active {
}

/* Theme */
.theme-dark {
}
```

## 7️⃣ 모던 CSS 기술

모던 CSS 기술은 최신 CSS 기능을 활용한 내용입니다.

### Flexbox 레이아웃

Flexbox는 1차원 레이아웃을 쉽게 구성할 수 있게 만들어줍니다.

```css
.flex-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}
```

### Grid 레이아웃

Grid는 단을 나누어 배치하는 데 유리한 방법입니다.

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
```

### 애니메이션과 트랜지션

애니메이션과 트랜지션은 요소의 움직임이나 크기, 상태 변화를 부드럽게 표현할 수 있게 해줍니다.

```css
.button {
  transition: all 0.3s ease;
}

@keyframes slide-in {
  from {
    transform: translateX(-100%);
  }
  to {
    transform: translateX(0);
  }
}
```

## 8️⃣ CSS 속성 상세

### 배경 관련 속성

배경 속성은 요소의 배경을 설정하는 데 사용됩니다.

```css
.background {
  /* 배경 축약형 */
  background: #fff url("image.jpg") no-repeat center/cover;

  /* 개별 속성 */
  background-color: #ffffff;
  background-image: url("image.jpg");
  background-position: center;
  background-size: cover;
  background-repeat: no-repeat;
}
```

### 테두리와 그림자

테두리와 그림자 속성은 요소의 외곽을 꾸미는 데 사용됩니다.

```css
.border-shadow {
  /* 테두리 */
  border: 1px solid #000;
  border-radius: 4px;

  /* 그림자 효과 */
  box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
}
```

### Flexbox 상세 속성

Flexbox는 요소의 배치를 유연하게 조정할 수 있게 해줍니다.

```css
.flex-container {
  display: flex;
  flex-direction: row; /* row, row-reverse, column, column-reverse */
  justify-content: space-between; /* flex-start, flex-end, center, space-around */
  align-items: center; /* stretch, flex-start, flex-end, baseline */
  flex-wrap: wrap; /* nowrap, wrap, wrap-reverse */
}

.flex-item {
  order: 1; /* 기본값 0, 정수값 */
  flex: 1; /* grow shrink basis의 축약형 */
  align-self: center; /* auto, flex-start, flex-end, center */
}
```

### Grid 상세 속성

Grid는 요소를 2차원으로 배치할 수 있게 해줍니다.

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
  grid-gap: 20px;
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
}

.grid-item {
  grid-area: header; /* 정의된 영역 이름 */
}
```

## 9️⃣ 고급 CSS 기능

### 트랜스폼과 트랜지션

트랜스폼과 트랜지션은 애니메이션의 변화를 부드럽게 표현할 수 있게 해줍니다.

```css
.transform {
  /* 2D 변환 */
  transform: rotate(45deg) scale(1.5) translate(50px, 100px) skew(30deg);

  /* 3D 변환 */
  transform: rotateX(45deg) rotateY(45deg) translateZ(100px);
  transform-origin: center;

  /* 트랜지션 */
  transition: all 0.3s ease;
  transition-property: transform;
  transition-duration: 0.3s;
  transition-timing-function: ease-in-out;
  transition-delay: 0.1s;
}
```

### 애니메이션 상세

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
    opacity: 0;
  }
  100% {
    transform: translateX(0);
    opacity: 1;
  }
}

.animated {
  animation: slide-in 0.5s ease-out forwards;
  animation-delay: 0.2s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
  animation-fill-mode: forwards;
  animation-play-state: running;
}
```

## 🔟 브라우저 호환성과 최적화

브라우저 호환성과 최적화는 다양한 브라우저에서 일관된 화면을 보여줍니다.

### 벤더 프리픽스

벤더 프리픽스는 특정 브라우저에서 지원하지 않거나 실험적인 CSS 속성을 사용해야 할 때 브라우저 접두사를 붙여 표현하는 것을 말해요!

```css
.prefix {
  -webkit-transform: rotate(45deg); /* Chrome, Safari */
  -moz-transform: rotate(45deg); /* Firefox */
  -ms-transform: rotate(45deg); /* IE */
  -o-transform: rotate(45deg); /* Opera */
  transform: rotate(45deg); /* 표준 */
}
```

### 미디어 쿼리 상세

미디어 쿼리는 다양한 장치와 화면 크기에 대응할 수 있도록 해주는 조건 구문입니다.

```css
/* 기기 해상도에 따른 대응 */
@media only screen and (min-resolution: 192dpi) {
  .high-res {
    background-image: url("high-res.jpg");
  }
}

/* 화면 방향 */
@media (orientation: landscape) {
  .rotate {
    flex-direction: row;
  }
}

/* 화면 비율 */
@media (aspect-ratio: 16/9) {
  .widescreen {
    max-width: 1200px;
  }
}
```

### CSS 변수 활용 예시

CSS 변수는 재사용 가능한 값을 정의하고 사용할 수 있게 합니다.

```css
:root {
  --main-color: #3498db;
  --secondary-color: #2ecc71;
  --spacing-unit: 8px;
  --max-width: 1200px;
}

.component {
  color: var(--main-color);
  padding: calc(var(--spacing-unit) * 2);
  max-width: var(--max-width);
}
```
